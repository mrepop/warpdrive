# WarpDrive

An iOS application for controlling and monitoring Warp console sessions from your iPhone or Android device.

## Features

- Remote session monitoring
- Terminal control and interaction
- Session management
- Real-time updates

## Requirements

- iOS 17.0+
- Xcode 26.0+
- Swift 5.9+

## Project Structure

```
WarpDrive/
├── WarpDrive/              # Main application code
│   ├── App/                # App lifecycle and configuration
│   ├── Views/              # SwiftUI views
│   ├── Models/             # Data models
│   ├── ViewModels/         # View models
│   ├── Services/           # Network and business logic
│   └── Resources/          # Assets and resources
├── WarpDriveTests/         # Unit tests
└── WarpDriveUITests/       # UI tests
```

## Getting Started

1. Clone the repository
2. Open `WarpDrive.xcodeproj` in Xcode
3. Build and run the project
